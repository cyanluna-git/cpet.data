# 피험자 편집 모달 전체 필드 추가

## 개요
피험자 편집 모달에 성별, 출생연도, 키, 체중 필드를 추가하여 모든 피험자 정보를 수정할 수 있게 개선했습니다.

## 주요 변경사항
- 추가한 것: 성별 선택 (남성/여성)
- 추가한 것: 출생연도 입력
- 추가한 것: 키(cm) 입력
- 추가한 것: 체중(kg) 입력
- 수정한 것: 모달 제목 "피험자 라벨 편집" → "피험자 정보 편집"
- 수정한 것: 모달에 스크롤 추가 (max-h-[60vh] overflow-y-auto)
