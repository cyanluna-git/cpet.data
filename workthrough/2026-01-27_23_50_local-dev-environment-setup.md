# 로컬 개발 환경 설정

## 개요
로컬 개발 환경에서 프론트엔드와 백엔드를 실행할 수 있도록 스크립트를 생성하고 포트 설정을 수정했습니다. Vite 프록시와 백엔드 포트 불일치 문제를 해결했습니다.

## 주요 변경사항
- **수정**: `backend/run.sh` 포트 8000 → 8100 (Vite 프록시와 일치)
- **추가**: `frontend/.env`에 명시적 포트 설정
- **추가**: `dev.sh` - 백엔드/프론트엔드 동시 실행 스크립트

## 핵심 코드

```bash
# dev.sh - 프로젝트 루트에서 실행
./dev.sh

# 또는 별도 터미널
cd backend && ./run.sh    # port 8100
cd frontend && npm run dev # port 3100
```

## 포트 구성
| 서비스 | 포트 |
|--------|------|
| Frontend | 3100 |
| Backend | 8100 |
| Database | 5100 |
